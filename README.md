# 🎬 Что глянуть?

Быстрый способ выбрать фильм или сериал на вечер без долгих споров и бесконечных списков.

**[👉 Посмотреть демо проекта](https://klirov.github.io/chto-glyanut/)**

---

## 🛠 Как это работает?
Приложение берет базу фильмов, применяет твои фильтры (жанр, рейтинг, год, топ-250) и выдает случайный вариант. 

* **Гибкость**: выбирай между фильмами и сериалами, настраивай рейтинг, годы и страну производства.
* **Приятный UX**: адаптивный дизайн, плавная анимация карточек и удобное управление.

## 🚀 Запуск локально
```bash
git clone https://github.com/klirov/chto-glyanut.git
cd chto-glyanut
npm install
npm run dev
```
Проект будет доступен по адресу http://localhost:4321/.

Сделано для того, чтобы ты тратил время на просмотр кино, а не на его поиск.
